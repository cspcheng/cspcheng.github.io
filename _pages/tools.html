TeXtidote

Have you ever thought of using a grammar checker on LaTeX files? TeXtidote solves this problem; it can read your original LaTeX file and perform various sanity checks on it: for example, making sure that every figure is referenced in the text, enforcing the correct capitalization of titles, etc. Try TeXtidote!
